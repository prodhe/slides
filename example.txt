slides

Example

Every paragraph is a new slide

- Monospaced
- Block centered
- Blanks at the beginning of a
  line will be preserved

Indent if you like:
	A tab
		equals
	four spaces

Special characters
	# comment
	@image.png

Escape with \
	\# comment
	\@image.png

Include images using @name.png

@r2d2_1.png

<b>HTML tags are escaped</b>

UTF-8 support (emojis!)
